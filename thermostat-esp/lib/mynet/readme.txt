Модуль выполняет 
- автоматическое АСИНХРОННОЕ подключение к сети WiFi и MQTT серверу 
- переподключение к сети в случае пропадания связи с WiFi 
- переподключение к MQTT серверув случае пропадания связи с WiFi 
- перепрошивку "OverTheAir" (OTA)

Для корректной работы модуля необходимо в функции setup вызвать функцию netInit 
и в функции loop вызывать netLoop()
Пример:
=========================================
#include <mynet.h>
#include "config.h"
void setup() {
  netInit(WIFI_SSID, WIFI_PASS, MQTT_SERVER, MQTT_CLIENT_ID);
}
void loop() {
  netLoop();
}
=========================================
Для работы OTA прошиваемый код должен также содержать вызовы netInit и netLoop
